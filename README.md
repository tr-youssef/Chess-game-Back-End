# Chess-game-Back-End

## Run the API

Clone the repository

```bash
git clone https://github.com/tr-youssef/Chess-game-Back-End.git
```

Install the dependencies

```bash
cd API
npm install
cd ..
cd Terminal
npm install
```

Create .env file in your API Folder

```bash
cd ..
cd API
touch .env
```

```python
PORT = 3000
DB_URL =  YOUR MONGODB CONNECTION URL
```

Run the application

```bash
cd API
npm start
cd ..
cd Terminal
npm start
```
